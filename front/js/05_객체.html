05_객체

1. 객체 선언

        - 중괄호{}를 사용해서 객체를 생성하고 속성에는 모든 자료형이 올 수 있다.
        - 객체는 속성명을 사용해서 속성을 식별한다.
        - 속성명에 공백이나 특수문자 등이 포함된 경우 따옴표로 묶어서 표기해야 한다.
        - 또한 객체에서 접근 시 무조건 대괄호를 사용해야 객체의 속성에 접근이 가능하다.

2. 객체의 메서드

        - 객체의 속성 중 함수 자료형인 속성을 메서드 라고 한다.

3. 객체와 반복문

        - for in : 모든 속성에 접근

4. 객체의 속성 추가 및 제거

        - delete : 'delete 객체.속성;'

5. 객체 배열

        -  비슷한 구조의 여러 개 객체를 모아놓은 배열로, 반복하면서 각 객체의 정보를 쉽게 다룰 수 있다.

6. 생성자 함수

        - new 연산자를 이용하여 객체 생성하는 '함수'
